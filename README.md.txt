1. This project is written in Typescript using Protractor tool, so node and npm are a prerequisite to run it.
2. To download all the dependencies defined in package.json: npm install
3. To run the e2e tests: Traverse to folder e2e and run the command: protractor conf.js
4. To check the reports after the tests: Traverse to e2e->report-> Copy the path of cucumber_report.html file and open
in browser to view the HTML reports.
